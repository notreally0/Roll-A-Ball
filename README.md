# Roll-a-Ball: Maze

This game is a customized version of the basic Unity Roll-a-Ball tutorial.

unlike the standard Roll-a-Ball tutorial, where there is no maze structure,
the player needs to move the yellow ball around the maze to pick up all the coins scattered within the maze corridors.

#Features
1. Custom maze.
2. 23 red Coins that need to be picked up from inside the maze.
3. UI system that calculates the score and displays "You Win!" when the player collects all the Coins.

#Controls
Move: W, A, S, D / Arrow keys
